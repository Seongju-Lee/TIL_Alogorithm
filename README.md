### 소개  
- 매일 한 문제씩 풀며 머리 굴리기     
- 성능 이슈를 대응하는 과정 속, 적절한 데이터 구조가 무엇인지 판단해야 할 때가 꽤 많다고 느낌        
- 병목의 원인이 되는 연산 과정을 파악하기 위해선 근본적인 문제 즉, 실행 원리에 대한 이해와 적절한 Complexity를 분석하는 능력의 필요성을 느낌        

때문에 매일 다양한 문제상황을 해결해보며, 다양한 알고리즘을 깊게 이해하고 기억하기 위해 꾸준히 노력
